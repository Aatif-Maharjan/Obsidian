## File Permission
- **7**: Read, write, and execute (4 + 2 + 1).
- **6**: Read and write (4 + 2).
- **5**: Read and execute (4 + 1).
- **4**: Read only (4).
- **2**: Write only (2).
- **1**: Execute only (1).
